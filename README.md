ブラウザ上でひらがな・漢字文字認識
==================================

## Requirement

* node.js
* npm


## Set up

```
$ npm install
```


## Usage

### Build (automatically)

```
$ npm start
```

* You can see the page in http://localhost:3000/

### Build (one shot)

```
$ npm run build
```

* Files are generated in `public`

### Release

```
$ npm run release
```

* Files are generated in `release`

### Clean up

```
$ npm run clean
```
